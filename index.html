Product Manual


Button A


The primary function of Button A is to recieve data from 2 pins and display the soil moisture level. If the soil moisture level is under 700, it is too dry and needs watering.


This button also has a second function as if you are in select mode to select the interval value for manually watering the plant. It will decrease (-) the value.

Button B


The primary function of Button B is to recieve data from the temperature sensor and display the temperature reading.


This button also has a second function as if you are in select mode to select the interval value for manually watering the plant. It will increase (+) the value.

Button A + B


The primary function of Button A + B is to toggle between auto and manual modes. Auto mode will automatically water the plant if the moisture level reading is under 700 and it will check every 5 seconds. Manual mode will water the plant at a set interval of time regardless of the moisture sensor, could be useful to control exactly how much water your need for your plant in a set interval.

Button Logo


The primary function of Button Logo is to toggle on select mode. In select mode, you can change the value for the interval of the manual watering. To decrease value, you press Button A. To increase value, you press Button B. Note that the interval is 10 seconds per set number. If you set it to 1, it will be 10 second interval.
